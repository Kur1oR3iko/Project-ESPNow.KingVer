# Project-ESPNow.KingVer
无线小纸条现以全新形态出击！这一次是为7789屏幕写的，所以如果你手持9341的屏幕，是 不 可 以 直接烧录的，这一点请注意

效果请见视频：https://www.bilibili.com/video/BV1ecQPBfEti/?vd_source=7cd787a16f1c8e304da1068dcebe2fdf

是个完全重写的版本，实现的功能如下
- 可调节线条粗细（2-12px）、彩虹色阶选择器、橡皮擦功能
- 截屏保存到 SD 卡（BMP 格式）、相册查看功能、图片删除功能
- 屏幕自动休眠（30秒/1分钟/5分钟/常亮）、收到绘画信息时自动唤醒
- 使用PlatformIO IDE进行编译，或者release压缩包解压直接烧录
