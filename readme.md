# ESP32 多人协同绘画板

## 项目介绍

这是一个基于 ESP32 开发的多人协同绘画板项目，支持多设备实时同步绘画内容，具有丰富的交互功能和用户体验。

## 功能特性

### 核心功能
- **多人实时绘画**：使用 ESP-NOW 协议实现多设备间绘画内容实时同步
- **丰富的绘画工具**：可调节线条粗细（2-12px）、彩虹色阶选择器、橡皮擦功能
- **作品管理**：截屏保存到 SD 卡（BMP 格式）、相册查看功能、图片删除功能
- **用户系统**：自定义用户名、在线用户列表
- **智能节能**：屏幕自动休眠（30秒/1分钟/5分钟/常亮）、收到绘画信息时自动唤醒
- **状态指示**：RGB LED 网络活动指示、在线用户数量显示

### 硬件支持
- **主控芯片**：ESP32 开发板
- **显示屏**：240x320 分辨率 ST7789 驱动 TFT 屏幕（直接支持）
- **触摸控制**：XPT2046 触摸控制器
- **存储**：SD 卡模块
- **指示灯**：RGB LED 三色指示灯

## 安装说明

### 硬件准备
- ESP32 开发板
- ST7789 驱动的 240x320 TFT 屏幕（可直接烧录）
- XPT2046 触摸模块
- SD 卡模块
- RGB LED（可选）

### 软件准备
- PlatformIO IDE
- Arduino 框架

### 烧录方法
1. 克隆本项目到本地
2. 使用 PlatformIO 打开项目
3. 连接 ESP32 开发板
4. 点击上传按钮进行烧录

### 屏幕兼容性
- **ST7789 屏幕**：直接烧录使用
- **ILI9341 屏幕**：需要修改 `platformio.ini` 文件中的屏幕驱动配置
